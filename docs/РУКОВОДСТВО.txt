РУКОВОДСТВО ПО ПРОЕКТУ FAKENEWS GENERATOR

=== ЧТО ТАКОЕ FAKENEWS GENERATOR ===

FakeNews Generator - это веб-приложение, которое позволяет создавать и просматривать фейковые новости. 
Приложение создано в образовательных и развлекательных целях.

=== ОСНОВНЫЕ ВОЗМОЖНОСТИ ===

1. СОЗДАНИЕ НОВОСТЕЙ
   - Выберите категорию (Политика, Технологии, Спорт, Развлечения)
   - Нажмите кнопку "Сгенерировать новость"
   - Готово! Новость создана и добавлена в общий список

2. ПРОСМОТР НОВОСТЕЙ
   - На главной странице отображаются все новости
   - Можно фильтровать новости по категориям
   - Нажмите на "Читать далее", чтобы увидеть полную новость

3. ВЗАИМОДЕЙСТВИЕ С НОВОСТЯМИ
   - Лайки: нажмите на сердечко, чтобы поставить лайк
   - Избранное: нажмите на закладку, чтобы добавить в избранное
   - Реакции: на странице новости можно добавить эмоциональную реакцию
   - Удаление: нажмите на крестик в правом верхнем углу карточки новости
   - Поделиться: используйте кнопки WhatsApp, Telegram или VK

4. АККАУНТ ПОЛЬЗОВАТЕЛЯ
   - Регистрация: создайте аккаунт, указав имя, email и пароль
   - Вход: войдите в систему, используя email и пароль
   - Профиль: просматривайте свои избранные новости и настройки

5. ДОПОЛНИТЕЛЬНО
   - Переключение темы: нажмите на иконку солнца/луны в шапке сайта
   - Статистика: на главной странице отображается статистика по новостям

=== КАК УСТРОЕН ПРОЕКТ ===

Проект построен на React.js и использует следующие технологии:
- React для создания пользовательского интерфейса
- CSS для стилизации
- localStorage для хранения данных в браузере
- Font Awesome для иконок
- React Router для навигации между страницами

Все данные хранятся локально в вашем браузере:
- Новости
- Информация о пользователях
- Настройки темы
- Избранные новости

=== СТРУКТУРА ПРОЕКТА ===

Проект состоит из следующих основных частей:

1. КОМПОНЕНТЫ
   - NewsCard: карточка новости
   - Header: шапка сайта с навигацией
   - Footer: подвал сайта

2. СТРАНИЦЫ
   - Home: главная страница со списком новостей
   - NewsDetail: страница с подробной информацией о новости
   - NewsGenerator: страница для создания новостей
   - Login/Register: страницы для входа и регистрации
   - Profile: страница профиля пользователя

3. КОНТЕКСТЫ И ХУКИ
   - NewsContext: управление новостями
   - AuthContext: аутентификация пользователей
   - ThemeContext: управление темой оформления

4. УТИЛИТЫ
   - Генераторы заголовков, содержания и изображений для новостей

=== КАК РАБОТАЕТ ГЕНЕРАЦИЯ НОВОСТЕЙ ===

Когда вы нажимаете кнопку "Сгенерировать новость":
1. Создается заголовок, соответствующий выбранной категории
2. Генерируется текст новости
3. Подбирается подходящее изображение
4. Создается объект новости с уникальным ID и текущей датой
5. Новость добавляется в общий список и отображается на странице

=== ШЕРИНГ НОВОСТЕЙ ===

Для того чтобы поделиться новостью:
1. Откройте страницу с новостью или сгенерируйте новую
2. Прокрутите вниз до раздела "Поделиться новостью"
3. Выберите нужную социальную сеть (WhatsApp, Telegram или VK)
4. Откроется окно для отправки сообщения с ссылкой на новость

=== УДАЛЕНИЕ НОВОСТЕЙ ===

Чтобы удалить новость:
1. Наведите курсор на карточку новости
2. Нажмите на крестик в правом верхнем углу
3. Подтвердите удаление в появившемся диалоговом окне
4. Новость будет удалена из списка и из избранного у всех пользователей

=== ТЕМЫ ОФОРМЛЕНИЯ ===

Приложение поддерживает светлую и темную темы:
1. Нажмите на иконку солнца/луны в правом верхнем углу
2. Тема изменится на противоположную
3. Ваш выбор сохранится для следующих посещений сайта
